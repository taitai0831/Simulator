How to move

  1. Gate Open.
  2. Gate Close.
  3. Lock lapber.
  4. Unlock Dispatch.
  5. If display "STATUSES" is "Yes", Dispatch.

Button

  • Gate : Gate Open,Close.
  • Dispatch : Start a vehicle
  • Unlock : Dispatch unlock.
  • Lapber : Lock the safety bar.

ATTENTION

Dispatch is performed once when the STATUSES is Yes.
